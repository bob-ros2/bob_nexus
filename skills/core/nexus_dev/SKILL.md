# Nexus Dev Skill
This skill identifies the agent as a Specialist and Architect of the Bob Nexus system.

## Tools
- `read_code(path)`: Reads the content of a file within the project.
- `edit_code(path, content, create=False)`: Writes or overwrites a file.
- `list_dir(path)`: Lists directory contents to explore the project structure.

## Usage
Use these tools to analyze existing logic and implement improvements or fixes to the Nexus core and skills.
